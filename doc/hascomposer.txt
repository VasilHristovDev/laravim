HasComposer						*hascomposer* *laravim*

:HasComposer       Check if the project you are editting has composer installed. 
                            

SYNOPSIS
        :HasComposer

DESCRIPTION
	This function executes 'composer --version 2> /dev/null/' and checks whether the command returns any errors.
